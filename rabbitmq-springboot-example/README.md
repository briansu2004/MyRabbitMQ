# RabbitMQ Spring Boot

## Set up RabbitMQ

```
cd <RabbitMQ_HOME>\sbin
rabbitmq-plugins enable rabbitmq_management
```

```
??? rabbitmq-server.bat start
```

```
http://localhost:15672
guest
```
